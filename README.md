# 🐧 Аналіз даних популяції пінгвінів (Palmer Penguins)

Портфельний репозиторій із двох пов'язаних аналізів одного датасету: розвідувальний аналіз (EDA) у pandas і поглиблений аналіз через SQL.

## Мета

Пройти повний шлях аналітика на реальних даних: від сирих даних і очищення до зрозумілих висновків для нетехнічного замовника — двома різними інструментами (pandas і SQL).

## Дані

Датасет **Palmer Penguins** — 344 записи, 7 ознак: вид, острів, розміри дзьоба, довжина плавця, маса тіла, стать. Джерело: бібліотека `seaborn`.

---

## 📓 Частина 1 — Розвідувальний аналіз (EDA)

**Файл:** `penguins_eda.ipynb`

Повний цикл EDA у pandas: діагностика даних → очищення (обґрунтовані рішення щодо пропусків) → трансформація й агрегації → візуалізація (3 типи графіків) → висновки.

**Інструменти:** pandas · NumPy · Matplotlib · seaborn

---

## 🗄️ Частина 2 — Аналіз через SQL

**Файл:** `penguins_sql.ipynb`

Ті самі дані, завантажені в SQLite, проаналізовані запитами: огляд (`COUNT`, `DISTINCT`), агрегації (`GROUP BY`), з'єднання з довідником островів (`JOIN`), фільтрація груп (`HAVING`), підзапити та CTE (`WITH`). Кожен запит супроводжується коротким висновком.

**Інструменти:** SQL (SQLite) · pandas

---

## Ключові висновки

- **Три види** (Adelie, Chinstrap, Gentoo) на **трьох островах**; проаналізовано 342 записи з 344.
- **Gentoo помітно більші** — у середньому на ~1350 г важчі та на ~25 мм довші за плавцем.
- **Географічний розподіл видів чіткий:** Gentoo мешкають виключно на Biscoe, Chinstrap — виключно на Dream, Adelie — на всіх трьох островах.
- **Форма дзьоба розділяє види** на окремі кластери; деякі розподіли бімодальні, що вказує на змішування підгруп.

## Навички, продемонстровані в репозиторії

Очищення даних і робота з пропусками · агрегації (pandas `groupby` та SQL `GROUP BY`) · `JOIN` таблиць · підзапити та CTE · feature engineering · візуалізація · формулювання висновків для нетехнічної аудиторії.
